# Resource location
This readme lists the binary resources that are (commonly) needed to run the project.

These resources are downloaded/generated automatically by CMake.

* [`ShakeNDry_1920x1080_120fps_420_8bit_YUV`](https://vaader-data.insa-rennes.fr/data/preesm/assets/ShakeNDry_1920x1080_120fps_420_8bit_YUV.zip): YUV video processed by the sobel filter.
* [`DejaVuSans.ttf`](https://preesm.github.io/assets/downloads/DejaVuSans.ttf): Font used in order to display fps information of the application.
